# GraphQL-Yoga + Vue3 simple CRUD

Simple CRUD example with GraphQL-Yoga + Vue 3 using Server Sent Events and subscriptions.

## Run the WebApp

On your terminal:

```
cd web<UiLibrayName>
npm install
npm run dev
```

## Run the server

On another terminal window/tab:

```
cd server
npm install
npx prisma migrate dev
npm run dev
```
